This plugin replaces the default WordPress post scheduler with Jalali post scheduler.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/block-editor-jalali-post-scheduler` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
